# Baseline reference dates for the dashboard

Policy-based reference dates used for “Time since implementation” and baseline comparisons.

**Acronym:** **ERP** = **Electronic Road Pricing** (Singapore’s current congestion charging system, introduced in 1998).

---

## London

| Label | Date | Description |
|-------|------|-------------|
| Congestion Charge start | 2003-02-17 | Central London Congestion Charge introduced. |
| Western Extension start | 2007-10-22 | CC zone extended west (Western Extension). |
| Western Extension removed | 2011-01-04 | Western Extension removed; zone back to original boundary. |
| Evening charge removed | 2011-01-02 | Charging hours reduced (evening charge removed). |
| Temporary suspension (COVID) | 2020-03-23 | CC suspended; resumed 2020-05-18. |
| Current charging hours | 2020-06-22 | Current scheme: 07:00–18:00 Mon–Fri, 12:00–18:00 Sat–Sun and bank holidays. |

---

## Singapore

| Label | Date | Description |
|-------|------|-------------|
| **ALS start** | 1975-06-02 | Area Licensing Scheme (ALS) started; world’s first congestion scheme. |
| **ERP start (Electronic Road Pricing)** | 1998-04-01 | ERP replaced ALS; main “start of ERP” reference. |
| **ERP 2.0 (new OBU)** | 2024-05-01 | New on-board units for new vehicles (technology rollout). |

*Note: Singapore annual traffic data in the app starts in 2004; ALS/ERP 1998 are for reference and “time since” only.*
